== Description ==

Want to try this plugin without installing it? Play arround on the [WP-SimpleViewer demo blog][demoblog]! 

This plugin enables you to easily add fancy [SimpleViewer][simpleviewer] Flash galleries to your posts and pages. The admin interface helps you to create the thumbs for a new gallery and change its settings. If you like your images can have captions that shows up in the gallery. All plugin and gallery settings can easily be changed on the admin pages.

Galleries can be added by selecting them with the "Add media" button of the plugin or straight by using a shortcode after the thumbs have been created in the backend: `[svgallery name="my_gallery"]`

For more information check the [WP-SimpleViewer support forum][suppforum]. The plugin comes along with [several translations][translations].

== Installation ==

1. Upload the plugin's complete directory into the `/wp-content/plugins/` directory. Simpleviewer is already included so there is no need to download it additionally.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on 'Settings' and select WP-SimpleViewerto check and adjust your settings
Then you will be able to create new galleries under 'Manage' and then 'WP-SimpleViewer galleries'. For more information check [this post][howto] on the support forum.

[howto]: http://wp-simpleviewer.fuggi82.de/topic/faq-or-how-to-use-the-plugin
            "WP-SimpleViewer support forum - FAQ/howto"

== Screenshots ==

1. The plugin's settings page
2. When everything is configured just upload your photos via ftp
3. Then create the gallery's files on the plugins page under 'Manage'. If you do not want to use the default values you can change them here, however the default values remain unchanged. 
4. You are able to edit the captions of the photos and settings of the gallery afterwards
5. To add the gallery select in from the 'Add Media' tab
6. Then you can see it in the post on your wordpress blog
